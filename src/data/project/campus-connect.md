## Overview

Campus Connect is a platform that helps BITS Pilani students find collaborators for projects, share skills, and connect with mentors.

## Problem

Students with complementary skills struggle to find each other. A developer needs a designer, a researcher needs an engineer, but there is no central place to connect.

## Why I Built It

I wanted to build a project but could not find a designer. After asking around, I realized this was a universal problem on campus.

## Process

1. Conducted user interviews with 30+ students
2. Designed the UX in Figma
3. Built the frontend with Next.js and TailwindCSS
4. Used Supabase for auth, database, and real-time features
5. Implemented a matching algorithm based on skills and interests

## Challenges

The cold start problem was real. Getting enough users for the platform to be useful required creative growth strategies.

## What I Learned

Community-driven products need a critical mass before they become useful. Start small, focus on one use case, and expand from there.

## Future Improvements

- Mobile app
- AI-powered project recommendations
- Integration with campus event systems
